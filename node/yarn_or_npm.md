# yarn、npm 比较

## 命令
|                    | npm            | yarn                                                                 |
| ------------------ | -------------- | -------------------------------------------------------------------- |
| init               | ```npm init``` | ```yarn init```                                                      |
| install dependency |                | ```yarn add [package]```                                     |
| install version    |                | ```yarn add [package]@[version]```<br>```yarn add [package]@[tag]``` |
| uninstall          |                |                                                                      |
| update             |                |                                                                      |
